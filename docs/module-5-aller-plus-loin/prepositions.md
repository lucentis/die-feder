# Les prépositions

**Prérequis :** [Le datif](./datif)

**Objectif :** savoir quel cas utiliser après les prépositions les plus courantes.

Chaque préposition impose un cas à ce qui la suit. Il y a trois groupes : prépositions toujours suivies de l'accusatif, toujours suivies du datif, et prépositions mixtes qui varient selon le sens.

::: tip Contractions courantes à connaître
*zu + dem → zum*, *zu + der → zur*, *in + dem → im*, *in + das → ins*, *an + dem → am*, *an + das → ans*. Ces contractions sont obligatoires dans la langue courante : on ne dit pas *zu dem Arzt* mais *zum Arzt*.
:::

::: warning bei vs zu : chez
*bei* = être chez quelqu'un (position) : *Ich bin bei Maria.* - Je suis chez Maria.
*zu* = aller chez quelqu'un (mouvement) : *Ich gehe zu Maria.* - Je vais chez Maria.
La distinction mouvement/position existe aussi pour les prépositions mixtes.
:::

## Prepositions mixtes (Wechselpräpositionen)

Ces prépositions peuvent prendre l'accusatif ou le datif selon qu'il y a mouvement ou position.

- **Accusatif** = mouvement vers (on répond a *wohin ?* - ou va-t-on ?)
- **Datif** = position, absence de déplacement (on répond a *wo ?* - ou est-on ?)

::: info L'astuce des verbes
Les verbes de mouvement vers un but (*gehen, legen, stellen, hangen, fahren...*) appellent l'accusatif. Les verbes d'etat ou de position (*sein, liegen, stehen, hangen, sitzen...*) appellent le datif. *Das Buch liegt auf dem Tisch* (datif, position) vs *Ich lege das Buch auf den Tisch* (accusatif, action de deposer).
:::
