# Le datif

**Prérequis :** [Expressions de temps](../module-4-le-temps/expressions-de-temps)

**Objectif :** former et utiliser le datif - le cas de l'objet indirect.

Le datif répond à la question "à qui ?" ou "pour qui ?". C'est le cas du bénéficiaire de l'action.

::: tip Le masculin et le neutre partagent la même forme au datif
*dem* pour les deux (défini), *einem* pour les deux (indefini). Seul le féminin est différent : *der* et *einer*. A noter : au datif, le féminin *der* ressemble au masculin nominatif - ne pas confondre.
:::

## Objet indirect : à qui ?

Quand une phrase a deux objets, le bénéficiaire de l'action (objet indirect) est au datif, et l'objet donné est à l'accusatif.

Autres exemples :

- *Er schreibt **seiner Mutter** einen Brief.* - Il ecrit une lettre a sa mere.
- *Sie zeigt **dem Mann** den Weg.* - Elle montre le chemin a l'homme.
- *Wir kaufen **unserem Freund** ein Geschenk.* - Nous achetons un cadeau a notre ami.

::: warning En allemand, le datif précède l'accusatif
Contrairement au français "je donne le livre a l'enfant" (COD avant COI), l'allemand place l'objet au datif avant l'objet a l'accusatif : *Ich gebe dem Kind das Buch* (datif, puis accusatif). C'est la règle lorsque les deux objets sont des noms.
:::

## Les pronoms personnels au datif

| Nominatif | Datif | Sens |
|-----------|-------|------|
| ich | **mir** | me / a moi |
| du | **dir** | te / a toi |
| er | **ihm** | lui |
| sie | **ihr** | lui (a elle) |
| es | **ihm** | lui (neutre) |
| wir | **uns** | nous |
| ihr | **euch** | vous |
| sie/Sie | **ihnen / Ihnen** | leur / a vous |

- *Kannst du **mir** helfen ?* - Tu peux m'aider ?
- *Ich gebe **dir** das Buch.* - Je te donne le livre.
- *Es gefallt **mir**.* - Ca me plait. (litt. "ca plait a moi")

## Verbes qui prennent le datif

Ces verbes exigent un complément au datif - ils n'ont pas d'objet direct.

| Verbe | Sens | Exemple |
|-------|------|---------|
| helfen | aider | *Ich helfe **dem** Mann.* |
| gefallen | plaire | *Berlin gefallt **mir**.* |
| gehoren | appartenir | *Das gehorte **meinem** Vater.* |
| danken | remercier | *Ich danke **dir**.* |
| antworten | répondre | *Antworte **dem** Lehrer.* |
| schmecken | avoir bon gout | *Das schmeckt **mir** gut.* |
| fehlen | manquer | *Du fehlst **mir**.* |

::: warning gefallen - la logique est inversee par rapport au français
*Berlin gefallt mir* signifie "Berlin me plait" - mais grammaticalement c'est "Berlin plait a moi" (*mir* = datif). Le sujet grammatical est Berlin, pas *mir*. Donc : *Berlin gefallt mir* (singulier), *Die Bucher gefallen mir* (pluriel) - le verbe s'accorde avec le sujet, pas avec *mir*.
:::

## Le pluriel au datif : ajout de -n

Au datif pluriel, les noms prennent un *-n* supplementaire s'ils n'en ont pas déjà un :

- *die Kinder* → *den Kinder**n*** - *Ich helfe den Kindern.* (J'aide les enfants.)
- *die Freunde* → *den Freunde**n*** - *Er dankt den Freunden.* (Il remercie les amis.)
- *die Autos* → *den Autos* (déjà en -s, pas de changement)
