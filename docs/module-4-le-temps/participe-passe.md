# Le participe passé

**Prérequis :** [Le Perfekt](./perfekt)

**Objectif :** former correctement le participe passé des verbes réguliers et connaître les irréguliers essentiels.

Le participe passé (Partizip II) est la deuxième pièce du Perfekt. Sa formation suit des règles prévisibles pour les verbes réguliers, mais les irréguliers sont à apprendre.

## Verbes réguliers : ge- + radical + -t

La formule est simple : on prend le radical (infinitif sans *-en*), on ajoute *ge-* devant et *-t* derrière.

::: info Radicaux en -t ou -d : on intercale un *-e-*
Même règle qu'au présent : *arbeiten* → *gearbeit**e**t*, *warten* → *gewart**e**t*. Le *-e-* est la pour faciliter la prononciation.
:::

## Deux cas où *ge-* n'apparaît pas

### Verbes en -ieren

Les verbes d'origine étrangère termines par *-ieren* ne prennent pas *ge-* : radical + *-t*.

- *telefonieren* → *telefoniert*
- *studieren* → *studiert*
- *passieren* → *passiert*
- *reservieren* → *reserviert*

### Verbes à préfixe non séparable

Les verbes avec préfixe non séparable (*be-, ge-, er-, ver-, zer-, ent-, emp-, miss-*) ne prennent pas *ge-* non plus : le préfixe + radical + *-t*.

- *bezahlen* → *bezahlt*
- *besuchen* → *besucht*
- *verkaufen* → *verkauft*
- *verstehen* → *verstanden* (irrégulier)

## Verbes séparables : ge- s'intercale

Pour les verbes séparables, le *ge-* s'insere entre le préfixe et le radical :

- *aufmachen* → *auf**ge**macht*
- *anrufen* → *an**ge**rufen*
- *einkaufen* → *ein**ge**kauft*
- *aufstehen* → *auf**ge**standen*

## Verbes irréguliers à connaître

Les irréguliers changent souvent la voyelle du radical et prennent *-en* au lieu de *-t*.

::: tip Apprendre par groupes de voyelle
Les irréguliers changent souvent la voyelle du radical de façon systématique. *trinken → getrunken*, *finden → gefunden*, *singen → gesungen* : le groupe *-in- → -un-* est courant. Apprendre par patterns plutot qu'un par un accelère la mémorisation.
:::
