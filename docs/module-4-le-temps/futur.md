# Le futur

**Prérequis :** [Le Präteritum](./prateritum)

**Objectif :** exprimer le futur en allemand - avec ou sans werden.

L'allemand a deux façons d'exprimer le futur. La plus simple est aussi la plus utilisée.

## Le présent à valeur de futur

En allemand, comme en français, le présent suffit souvent à exprimer le futur quand le contexte temporel est clair. C'est la forme la plus naturelle à l'oral.

- *Ich fahre morgen nach Berlin.* - Je vais a Berlin demain.
- *Er kommt um 8 Uhr.* - Il arrive a 8h.
- *Wir essen heute Abend im Restaurant.* - Ce soir on mange au restaurant.

::: tip Le présent + indicateur de temps = futur
Un mot comme *morgen* (demain), *heute Abend* (ce soir), *nachste Woche* (la semaine prochaine) suffit a signaler que la phrase parlé du futur. Dans ce cas, le présent est preferable a werden dans la conversation.
:::

## werden + infinitif : le futur formel

*Werden* conjugué + infinitif en fin de phrase. Cette structure s'utilisé pour des prédictions, des promesses ou pour insister sur le caractère futur de l'action.

### Conjugaison de werden (rappel)

| Pronom | werden |
|--------|--------|
| ich | **werde** |
| du | **wirst** |
| er/sie/es | **wird** |
| wir | **werden** |
| ihr | **werdet** |
| sie/Sie | **werden** |

### Exemples avec werden

- *Es wird morgen regnen.* - Il va pleuvoir demain. (prediction meteo)
- *Du wirst das verstehen.* - Tu vas comprendre ca.
- *Wir werden sehen.* - On verra bien.
- *Das wird gut sein.* - Ca va être bien.

::: warning werden peut aussi exprimer la probabilite au présent
*Er wird krank sein.* peut signifier "il doit probablement être malade" (supposition sur le présent), pas seulement "il sera malade" (futur). Le contexte tranche. Cette nuance est avancee - a noter pour plus tard.
:::
