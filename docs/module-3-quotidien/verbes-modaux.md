# Les verbes modaux

**Prérequis :** [Poser des questions](./poser-des-questions)

**Objectif :** utiliser les six verbes modaux pour exprimer la capacite, l'obligation, la permission et la volonte.

Les verbes modaux (*Modalverben*) modifient le sens du verbe principal. Ils se conjuguent seuls et repoussent le verbe principal a l'infinitif en fin de phrase.

## La structure avec un modal

Le modal est conjugue en position 2. Le verbe principal reste a l'**infinitif en fin de phrase**.

Autres exemples :

- *Du musst jetzt **schlafen**.* - Tu dois dormir maintenant.
- *Er will nach Berlin **fahren**.* - Il veut aller a Berlin.
- *Wir dürfen hier nicht **parken**.* - On n'a pas le droit de se garer ici.
- *Soll ich das **machen** ?* - Je suis sense faire ca ?

::: warning können vs dürfen - pouvoir n'est pas toujours "können"
*Können* = capacité physique ou intellectuelle (*ich kann schwimmen* - je sais nager).
*Durfen* = permission (*ich darf hier parken* - j'ai le droit de me garer ici).
En français, « pouvoir » couvre les deux. En allemand, la distinction est obligatoire.
:::

## möchten - voudrais (forme polie de mögen)

*Möchten* est la forme de politesse pour exprimer un souhait, très utilisée pour commander ou demander :

| Pronom | möchten |
|--------|---------|
| ich | möchte |
| du | möchtest |
| er/sie/es | möchte |
| wir | möchten |
| ihr | möchtet |
| sie/Sie | möchten |

- *Ich möchte einen Kaffee.* - Je voudrais un cafe.
- *Wir möchten bitte die Rechnung.* - Nous voudrions l'addition, s'il vous plait.

::: tip möchten est plus poli que wollen
*Ich will einen Kaffee* est grammaticalement correct mais sonne brusque. En situation réelle, utilisé toujours *ich möchte* pour commander ou demander quelque chose. C'est l'équivalent de "je voudrais" vs "je veux".
:::
