# Poser des questions

**Prérequis :** [Vocabulaire du quotidien](./vocabulaire-quotidien)

**Objectif :** former des questions ouvertes avec les mots interrogatifs allemands.

Les mots interrogatifs allemands commencent presque tous par *W-*, d'où leur nom : les *W-Fragen*.

::: warning wo / wohin / woher - trois mots pour "ou"
C'est le point le plus déroutant pour les francophones. L'allemand distingue le lieu statique (*wo* - où est-ce ?), la destination (*wohin* - vers où ?) et l'origine (*woher* - d'où ?). Cette distinction est obligatoire et change le verbe utilisé : *sein* pour *wo*, *gehen/fahren/kommen* pour *wohin/woher*.
:::

## Structure de la W-Frage

La règle V2 s'applique : le mot interrogatif est en position 1, le verbe en position 2.

## wer se décline aussi

*Wer* (qui) se décline comme le masculin singulier pour le cas :

- *Wer* = nominatif : *Wer kommt?* - Qui vient ?
- *Wen* = accusatif : *Wen rufst du an?* - Tu appelles qui ?
- *Wem* = datif : *Wem gibst du das?* - A qui tu donnes ca ?

::: tip welch- suit la déclinaison des articles
*Welcher* (masculin nom.), *welche* (féminin), *welches* (neutre), *welchen* (masculin acc.) - la terminaison change comme pour *ein/eine*. Dans le doute, utilisé *welche* qui fonctionne pour le féminin et le pluriel.
:::
