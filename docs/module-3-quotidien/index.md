# Module 3 — Le quotidien

**Prérequis :** [Module 2 - Construire une phrase](../module-2-construire-une-phrase/)

**Objectif :** parler de sa vie et de ses habitudes, poser des questions, comprendre et être compris dans des situations courantes.

Tu sais construire une phrase simple. Dans ce module, on enrichit le vocabulaire et on introduit l'accusatif - le cas de l'objet direct - qui ouvre beaucoup de nouvelles possibilités d'expression.

::: info Ce module finalise les bases grammaticales du quotidien
Accusatif, verbes séparables, modaux - ce sont les trois blocs qui te permettront de tenir une vraie conversation simple. Les situations ciblees sont concretes : commander, se deplacer, parler de ses habitudes.
:::
