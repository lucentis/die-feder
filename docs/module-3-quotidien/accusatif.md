# L'accusatif

**Prérequis :** [La négation](../module-2-construire-une-phrase/negation)

**Objectif :** reconnaitre et utiliser l'accusatif - le cas de l'objet direct.

Le nominatif, vu au module 2, est le cas du sujet. L'accusatif est le cas de l'objet direct - ce qui subit l'action.

## Sujet et objet direct

Dans *"Le chat mange la souris"*, le chat fait l'action (sujet, nominatif) et la souris la subit (objet direct, accusatif).

- *Der Mann kauft **einen Apfel**.* - L'homme achete une pomme.
- *Das Kind liebt **die Katze**.* - L'enfant aime la chatte. (neutre grammatical → die Katze, féminin)
- *Ich lese **das Buch**.* - Je lis le livre.

## Ce qui change à l'accusatif

Bonne nouvelle : seul l'article masculin change. Féminin, neutre et pluriel restent identiques au nominatif.

::: tip Seul le masculin change
*der* → *den* et *ein* → *einen*. C'est le seul changement a retenir pour l'accusatif. Féminin, neutre et pluriel sont identiques au nominatif.
:::

## Identifier l'objet direct

L'objet direct répond à la question "quoi ?" ou "qui ?" après le verbe :

- *Ich kaufe **einen Kaffee**.* - J'achete **quoi** ? Un cafe. → accusatif masculin
- *Er sieht **die Frau**.* - Il voit **qui** ? La femme. → accusatif féminin (inchange)
- *Wir brauchen **das Auto**.* - Nous avons besoin de **quoi** ? La voiture. → accusatif neutre (inchange)

## kein à l'accusatif

*Kein* suit la même logique : seul le masculin change.

- *Ich habe **keinen** Hunger.* - Je n'ai pas faim. (masculin)
- *Er hat **keine** Zeit.* - Il n'a pas le temps. (féminin, inchange)

::: warning Les pronoms changent aussi
Les pronoms personnels ont également des formes accusatif : *mich* (me), *dich* (te), *ihn* (le), *sie* (la), *es* (le/la neutre), *uns* (nous), *euch* (vous), *sie* (les). Ils seront detailles progressivement dans les prochaines lecons.
:::

## Verbes qui appellent l'accusatif

Ces verbes très courants prennent toujours un objet direct à l'accusatif :

| Verbe | Sens | Exemple |
|-------|------|---------|
| kaufen | acheter | *Ich kaufe einen Apfel.* |
| essen | manger | *Er isst einen Kuchen.* |
| trinken | boire | *Sie trinkt einen Tee.* |
| sehen | voir | *Wir sehen den Film.* |
| lesen | lire | *Du liest das Buch.* |
| haben | avoir | *Ich habe einen Bruder.* |
| brauchen | avoir besoin de | *Ich brauche einen Stift.* |
