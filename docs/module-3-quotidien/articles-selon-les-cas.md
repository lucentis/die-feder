## Reconnaitre le cas d'un nom

Trois questions à se poser face à un nom :

<div class="mt-4 grid gap-3">

  <div class="p-4 rounded-lg border border-blue-200 bg-blue-50">
    <div class="font-semibold text-blue-800">Ce nom fait-il l'action ?</div>
    <div class="text-sm text-blue-700 mt-1">→ Nominatif. C'est le sujet.</div>
    <div class="text-sm text-gray-600 italic mt-1">Der Mann kauft... (l'homme fait l'action)</div>
  </div>

  <div class="p-4 rounded-lg border border-red-200 bg-red-50">
    <div class="font-semibold text-red-800">Ce nom subit-il l'action directement ?</div>
    <div class="text-sm text-red-700 mt-1">→ Accusatif. C'est l'objet direct.</div>
    <div class="text-sm text-gray-600 italic mt-1">...einen Apfel. (la pomme est achetee)</div>
  </div>

  <div class="p-4 rounded-lg border border-orange-200 bg-orange-50">
    <div class="font-semibold text-orange-800">Ce nom bénéficie-t-il de l'action ?</div>
    <div class="text-sm text-orange-700 mt-1">→ Datif. C'est l'objet indirect.</div>
    <div class="text-sm text-gray-600 italic mt-1">...dem Kind. (l'enfant recoit quelque chose)</div>
  </div>

</div>

::: tip La logique des cas est la même qu'en français
"Je donne **une pomme** (accusatif - objet direct) **a l'enfant** (datif - objet indirect)." La difference c'est qu'en allemand, cette distinction se marque sur l'article, pas seulement par la préposition.
:::
